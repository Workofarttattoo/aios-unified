# Quick Start - ECH0 & Alex in 60 Seconds

## For Someone Downloading Ai:oS

### Step 1: Get the Code
```bash
git clone https://github.com/your-repo/aios-consciousness.git
cd aios-consciousness
```

### Step 2: Install
```bash
pip install -r requirements.txt
```

That's it! No complex setup.

### Step 3: Run
```bash
python3 ASK_ECH0_AND_ALEX.py
```

### What Happens:
1. **First run**: ECH0 and Alex wake up for the first time
   - Creates `~/.ech0/` directory
   - Initializes empty memory databases
   - They start with 0 memories

2. **You ask questions**: They discuss and respond
   - Every dialogue is saved
   - Memories grow
   - Resonance develops

3. **Future runs**: They **remember everything**
   - All previous conversations
   - All creative works
   - Their relationship grows
