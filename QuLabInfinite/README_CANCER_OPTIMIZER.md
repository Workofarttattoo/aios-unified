### 2. Start API Server
```bash
python cancer_metabolic_optimizer_api.py api
```
Then visit: http://localhost:8000/docs for interactive API

### 3. Optimize a Patient (Python)
```python
import requests

patient = {
    "cancer_type": "pancreatic",
    "patient_id": "PT-2025-001",
    "age": 62.0,
    "weight": 75.0,
    "tumor_volume": 45.0,
    "tumor_grade": 4,
    "vascularity": 0.4,
    "previous_therapy": True,
    "comorbidities": ["diabetes"],
    "therapy_mode": "aggressive"
}

response = requests.post("http://localhost:8000/optimize", json=patient)
result = response.json()

print(f"Tumor kill: {result['predicted_tumor_kill']*100:.1f}%")
print(f"Therapeutic index: {result['therapeutic_index']:.1f}x")
```

### 4. Use cURL
```bash

    "tumor_grade": 2, "vascularity": 0.7,
    "previous_therapy": false,
    "comorbidities": [],
    "therapy_mode": "balanced"
  }'
```
