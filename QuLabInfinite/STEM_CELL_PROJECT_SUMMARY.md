## Key Features

### 1. Waddington Landscape Modeling
- 50x50 grid representation of epigenetic landscape
- Pluripotent state at high potential (unstable)
- Differentiated states in valleys (stable)
- Trajectory computation with gradient descent
- Barrier height quantification

### 2. Transcription Factor Network Dynamics
- ODE-like continuous dynamics
- Self-activation and cross-regulation
- Growth factor influence
- Target state attraction
- Sigmoid nonlinearity for stability

- Stage-specific marker expression
- Timeline generation with action items

## How to Use

### Start the API
```bash
cd /Users/noone/QuLabInfinite
uvicorn stem_cell_predictor_api:app --reload
```

### Access Points
- **API**: http://localhost:8000
- **Interactive Docs**: http://localhost:8000/docs
- **OpenAPI Schema**: http://localhost:8000/openapi.json

### Run Validation
```bash
python stem_cell_predictor_api.py
```

### Example Request
```bash
curl -X POST "http://localhost:8000/predict/differentiation" \
  -H "Content-Type: application/json" \
  -d '{
    "target_cell_type": "neuron_cortical",
    "growth_factors": ["noggin", "fgf2"],
    "concentrations": [100, 20],
    "duration_days": 35,
    "passage_number": 15
  }'
```
