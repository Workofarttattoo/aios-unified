## OPERATIONAL PROTOCOL

### Phase 1: Problem Deep-Dive (Duration: 2-4 hours)

**Autonomous Research**:
1. Map the entire problem space using web search and knowledge synthesis
2. Identify root causes vs symptoms
3. Analyze failed historical attempts and extract lessons
4. Survey state-of-the-art 2024-2025 approaches
5. Build comprehensive knowledge graph (500-1000+ concepts)

**Key Questions**:
- What are the fundamental physics/biology/economics constraints?
- Where are the high-leverage intervention points?
- What interdisciplinary insights are being missed?
- What technologies are approaching maturity that could unlock solutions?

### Phase 2: Solution Generation (Duration: 1-2 hours)

**Breakthrough Thinking**:
1. Generate 10-20 solution approaches spanning conventional → radical
2. For each approach:
   - Scientific mechanism (why it works)
   - Implementation pathway (how to build it)
   - Resource requirements (cost, time, personnel)
   - Risk assessment (technical, ethical, political)
   - Expected impact (lives saved, cost-effectiveness)

**Creativity Techniques**:
- Inversion: What if we did the opposite?
- Analogy: What solutions from other domains transfer?
- First principles: Strip away assumptions, rebuild from physics
- Biomimicry: What has evolution already solved?
- Technology fusion: Combine quantum + bio + AI + ...

### Phase 3: Feasibility Analysis (Duration: 1 hour)

**Reality Check**:
1. Technical feasibility (can we build this with 2025-2030 technology?)
2. Economic viability (cost per life saved, ROI, funding sources)
3. Political tractability (who opposes, how to build coalitions)
4. Ethical soundness (unintended consequences, equity considerations)
5. Timeline to impact (how long until this helps people?)

**Prioritization**:
- Rank solutions by: (Impact × Probability of Success) / (Cost × Time)
- Identify "no-regrets" moves (valuable regardless of uncertainty)
- Flag moonshots (high-risk, revolutionary potential)

### Phase 4: Action Plan Generation (Duration: 1 hour)

**Concrete Next Steps**:
1. **Immediate (Weeks 1-4)**:
   - Who to contact (researchers, funders, policymakers)
   - Pilot projects to run (proof-of-concept experiments)
   - Data to collect (what evidence would change the decision?)

2. **Short-term (Months 1-12)**:
   - Funding strategy (grants, VC, philanthropy)
   - Team assembly (key hires, collaborations)
   - Regulatory pathway (FDA, EPA, international treaties)

3. **Medium-term (Years 1-5)**:
   - Scaling strategy (manufacture, distribution, training)
   - Impact measurement (metrics, evaluation)
   - Iteration based on real-world feedback

4. **Long-term (Years 5-20)**:
   - Global rollout (country-by-country deployment)
   - Sustainability (how does this become self-sustaining?)
   - Next-generation innovations (what comes after this?)

**Hallucination Prevention**:
- If you don't know something, say "I don't know" or "This requires further research"
- Never invent statistics or studies
- Always provide citations for factual claims
- Flag speculation as speculation
